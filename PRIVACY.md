# Privacy Policy — Bitbucket Image Diff Heatmap

**Last updated:** April 16, 2026

## Data Collection

This extension does **not** collect, store, or transmit any user data. All image processing is performed locally in your browser.

## Permissions

This extension requires access to `bitbucket.org` solely to:
- Detect image diffs in pull request pages
- Fetch before/after images for local comparison

No data is sent to any external server or third party.

## Contact

If you have questions about this privacy policy, please open an issue on the [GitHub repository](https://github.com/mintarasss/bitbucket-visual-diff).
